The Problem

Renal Cell Carcinoma — The Clinical Challenge

RCC: Why It Matters

The Numbers

  • 400,000 new cases/year worldwide
  • 3rd most common urological cancer
  • Metastatic 5-year survival <15%
  • Male:Female incidence 2:1

The Paradox

  • Males get it more often
  • But females respond better to immunotherapy
  • Why? Hormones? Genetics? Metabolism?
  • Traditional models can't answer this

Why Agent-Based Modeling?

ABM Strengths

  • Heterogeneous cell populations
  • Spatial dynamics & local interactions
  • Emergent behavior from simple rules
  • Stochastic — matches biological variability

ODE/PDE Limitations

  • Assume well-mixed populations
  • No cell-level heterogeneity
  • No spatial resolution
  • Can't model discrete events

Project Objectives

1. Port Mesa → Repast4Py (MPI-based, 3D)
2. Add glucose microenvironment — 3D reaction-diffusion, Warburg effect
3. Validate against ARON-1 clinical data
4. Bayesian optimization — 92 weight parameters with Optuna TPE
5. Sex-specific analysis — uncover why females respond better

The Model

18 Agent Types · 3D Grid · Glucose Field

Agent Hierarchy — 18 Types

Design Choices

  • AgentType(IntEnum) — O(1) dispatch
  • Hierarchical inheritance for shared behavior
  • MPI-safe serialization (no class-level state)
  • spatial_index dict for O(1) lookups

Immune System Overview

Innate Immunity

  • M1 Macrophages — phagocytose tumor, pro-inflammatory
  • M2 Macrophages — tissue repair, pro-tumor
  • NK Cells — MHC-independent killing
  • Dendritic / pDC — antigen presentation, IFN
  • Mast / Neutrophil — early response, angiogenesis

Adaptive Immunity

  • CD8+ CTL — direct tumor killing via TCR
  • Th1 — promotes CTL and M1 activity
  • Th2 — humoral response branch
  • Treg — suppresses immune activation
  • PD-1/PD-L1 checkpoint modulation

Key Systems

DNA · Treatment · Sex Hormones

DNA Mutation Engine

16 Real Genes

  • Full nucleotide sequences (~3200 nt total)
  • TATA box + coding region + stop codon
  • 64-codon translation table → 20 amino acids
  • Point mutations during cell division
  • Levenshtein distance for protein similarity

5 Phenotypic Effects

  • Antigen presentation — MHC, JAK, B2M, TAP
  • Checkpoint inhibition — CD274 (PD-L1)
  • Genomic instability — BRCA1
  • Tumor suppression — TP53, RB1, CDKN2A
  • Extra proliferation — MYC, RAS, EGFR, HER2

Sex difference: males start with 3 BRCA1 mutations vs 1 for females

Treatment System

ICI — Immune Checkpoint Inhibitor

  • Blocks PD-1/PD-L1 axis
  • Restores T cell activation
  • Increases immune infiltration

TKI — Tyrosine Kinase Inhibitor

  • Reduces tumor growth rate
  • Blocks angiogenesis
  • Releases neoantigens

ICI + TKI Combination

TKI weakens tumor defenses → ICI delivers the killing blow
Proportional averaging: each drug gets 1/N effect strength

Sex Hormone System

Three Hormones

Estrogenweight 0.45
Testosteroneweight 0.30
Progesteroneweight 0.25

Each modulates 17 effect channels: Th1/Th2 proliferation, IFN-γ, PD-1, MHC, kill rate…

CD8+ Sex Drift

  • Female: −drift (fewer naïve T cells)
  • Male: +drift (more naïve T cells)
  • Max drift 25% over 15 steps

Combined with hormone-mediated Th1 boosting in females → stronger anti-tumor response

3D Grid & Glucose Field

SharedGrid

  • 3D MPI-distributed, sticky borders
  • Multiple agents per cell
  • Moore neighborhood (26-connected)
  • Deferred movement pattern

Glucose Cycle

  1. Blood vessels inject
  2. scipy.ndimage.convolve diffuses
  3. Natural decay: C ← C(1−λ)
  4. Cells consume locally

Gradient ∇C drives immune chemotaxis

92 Tunable Parameters & Bayesian Optimization

Parameter Space

  • 84 original weight parameters (62 log-uniform, 15 biases…)
  • 8 new glucose parameters (diffusion, decay, consumption…)
  • = 92 tunable weights optimized by Optuna
  • Plus 4 model + 18 patient params (fixed per simulation)

Optuna TPE

  • 250 trials — Tree-structured Parzen Estimator
  • Target: ARON-1 clinical survival data
  • Objective: MSE with 500² penalty for outcome mismatch
  • Output: best parameter JSON + convergence curve
